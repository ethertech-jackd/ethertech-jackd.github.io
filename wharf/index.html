WHARF Overview

WHARF is the backend inference engine and operator console behind ATAKRR. It enables dataset coordination, deferred task queuing, signal ingest and geolocation processing, and integration with remote Fog Nodes. Built for asynchronous RF workflows and ground-truth operator workflows.
