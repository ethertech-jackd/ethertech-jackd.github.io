ATAKRR: ATAK Radio Reconnaissance

Passive Wideband RF AMC RF Fingerprinting ATAK / CoT Geolocation

Overview

ATAKRR is an open-source R&D platform for passive spectrum monitoring, automatic modulation classification (AMC), and RF fingerprinting, built to integrate directly with ATAK (Android Team Awareness Kit). The goal: real-time, team-shared electromagnetic situational awareness.

We’re actively pursuing grants (DARPA submission made) and collaborators. Android/ATAK developer? RF/ML? Reach us: atakrr (a@t) ethertech.org. More background: ATAKRR on Notion.

Features

  • Passive wideband monitoring (1 MHz–6 GHz with HackRF)
  • Deep learning AMC (CNN / RNN / Transformer / DBN / GNN)
  • RF fingerprinting (device-level ID)
  • Geolocation: triangulation/trilateration with synchronized SDRs; single-device “fox-hunting” DF
  • ATAK integration: CoT overlays, heatmaps, markers, device tracking
  • Dataset tools: preprocessing, augmentation, few-shot/meta-learning

Why ATAKRR?

We live inside the EM spectrum. Humans see visible light; the rest is invisible. ATAKRR makes RF “visible” and understandable: Wi-Fi vs voice vs jammer, plus unique device fingerprints — type, encoding (“language”), frequency (“color”), and when possible, location.

In ATAK, you’ll see overlays of transmitters, relationships, movement over time, and signals that indicate gatherings or anomalies. It’s an RF map of the world that’s been hidden in plain sight.

How it Works

Hardware / Signal Collection Layer

  • HackRF view from 1 MHz–6 GHz @ 20 MSps (~20 MHz BW sweep)
  • Other radios (FPGA/ASIC codecs) for protocol specifics (e.g., Wi-Fi/BLE MACs)
  • Multi-node sync → shared FoG compute (“The Mothership”) for geolocation math

Signal Processing / AI Layer

  • Sweep + waterfall via hackrf_sweep-style scanning
  • AMC via CNN/RNN/Transformer; wavelet/CWT and attention for low-SNR
  • RF fingerprinting for unique device identification

Visualization Layer (ATAK)

  • Overlays: positions, trails, fingerprints, device clusters
  • CoT markers pipe into the wider TAK ecosystem for team awareness

ATAK Plugin Layer

  • Heatmaps, signal IDs, threat markers
  • TAK server integration for shared spectrum picture
  • Hooks for FoG / BAMMO / PEWPEW modules

Dataset + Training Tools

  • GNU Radio/RadioML capture → preprocessing
  • Augmentation for multipath, fading, low-SNR
  • Few-shot/meta-learning for rare emitters

What You Get

  • ATAK map overlay of detected transmitters
  • Movement tracking and device relationships
  • Frequency ↔ MAC/identifier correlation
  • Signals-of-interest to infer gatherings or anomalies
